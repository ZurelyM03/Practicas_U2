Web 1.0

Apareció hacia 1990 y es muy primitiva para lo que hoy ofrece la web.

Web 2.0

Apareció en 2004 y contiene los foros, blogs, comentarios y después las redes sociales a demas permite compartir información.

Web 3.0

Aparecio en 2006 está relacionada con la web semántica, y su principal objetivo es democratizar el entorno digital.

Web 4.0

Aparecio en 2016, es una capa de integración necesaria para la explotación de la Web semántica y sus enormes posibilidades.
